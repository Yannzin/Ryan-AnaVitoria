# Ryan-AnaVitoria


Exercício 1 - Criando Políticas de Segurança com IAM
Prazo: 2 semanas (entrega na Aula 03)

Passo a passo:  
1º Crie uma conta na AWS: aws.amazon.com/free  
2º Acesse o painel do IAM  
3º Crie 2 usuários: aluno-admin e aluno-leitor  
4º Crie 2 grupos: Administradores e SomenteLeitura  
5º Atribua as políticas AdministratorAccess e ReadOnlyAccess  
6º Adicione cada usuário ao grupo correspondente
